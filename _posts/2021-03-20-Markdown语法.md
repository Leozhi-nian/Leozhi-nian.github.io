# Epsilon Notes Markdown语法

## Markdown标题

### 1. 类Setext形式

###### 语法格式

```markdown
这是标题一
==========

这是标题二
----------
```

###### 显示效果

这是标题一
==========

这是标题二
----------

### 2. 类Atx形式

###### 语法格式

```markdown
# 这是标题一
## 这是标题二
### 这是标题三
#### 这是标题四
##### 这是标题五
###### 这是标题六
```

###### 显示效果

# 这是标题一
## 这是标题二
### 这是标题三
#### 这是标题四
##### 这是标题五
###### 这是标题六

###### 语法格式


###### 语法格式

### 9. 脚注

###### 语法格式


###### 显示效果

这是脚注[^1]：点击后会跳转至注解部分

## Markdown区块

### 1. 区块引用

###### 语法格式

```markdown

```

###### 显示效果

### 2. 嵌套引用

###### 语法格式

```markdown
> 第一层引用
>> 第二层引用
>>> 第三层引用
>>>> 第四层引用
>>>>> 第五层引用
>>>>>> 第六层引用
```

###### 显示效果

> 第一层引用
> > 第二层引用
> > > 第三层引用
> > > > 第四层引用
> > > > > 第五层引用
> > > > >
> > > > > > 第六层引用

### 3. 区块嵌套列表

###### 语法格式

```markdown
> 区块中嵌套列表
> 1. 第一项
> 2. 第二项
> * 第一项
> * 第二项
```

###### 显示效果

> 区块中嵌套列表
> 1. 第一项
> 2. 第二项
> * 第一项
> * 第二项

### 4. 列表嵌套区块

###### 语法格式

```markdown

    > 列表内区块引用
* 第二项
    > 列表内区块引用
```

  > 列表内区块引用
